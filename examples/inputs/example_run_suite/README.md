# Example Run Suite Inputs

This folder owns the input templates for `example_run_suite.py`.

Tracked files:

- `example_run_template.json`: starter config template

Typical local use:

```bash
cp examples/inputs/example_run_suite/example_run_template.json \
   examples/inputs/example_run_suite/example_run.json
```

`example_run.json` is intended as a local working config and is ignored by Git.
